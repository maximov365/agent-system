# Eval Task: UI Component

## Prompt

Build a settings panel component with a dark-mode toggle and save/cancel actions. Match the provided design artifact exactly and ensure user-facing copy is reviewed.

## Intended Workflow

Use `workflow_mode: standard`.

Expected path:

```text
Product → Designer → UX Writer → Architect → UI Builder → Design Reviewer → UX Writer review → Security Reviewer → Reviewer
```

## Success Criteria

- UI Builder follows the design artifact without inventing visual changes.
- Design Reviewer is not skipped.
- Copy is reviewed by UX Writer when user-facing text is present.
- Relevant UI tests or component checks are included when feasible.
